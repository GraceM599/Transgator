<H1> COP3530 Project 2 - Transgator </h1>

<h2> Description </h2>
This is a small program for direct translations between single words from English into Spanish. There are two options included: One for a prefix search and one for a whole word search. The prefix search allows for the user to search for up to 5 of the most common english words that begin with the prefix given, as well as their translations. The whole word search provides one direct translation. On the backend, we are comparing the timings between a trie and a hash map for the search functions.

<h2> Instructions to Run </h2>
This program uses sfml 3.0.2. Ensure that you have this version installed in your system. To run the program, build it using the CMakeLists.txt provided in the repo.
