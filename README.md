<H1> COP3530 Project 2 - Transgator </h1>

<h2> Description </h2>
This is a small program for direct translations between single words from English into Spanish. There are two options included: One for a prefix search and one for a whole word search. The prefix search is a variation of autocomplete where the user will search for a prefix and the program will display up to 5 of the most common english words with that prefix along with their translations. The whole word search provides one direct translation. On the backend, we are comparing the timings between a trie and a hash map for the search functions.

<h2> Instructions to Run </h2>
This program uses SFML 3.0.2. To run the program, ensure that you have the correct SFML version installed to your system and build the program using the CMakeLists.txt provided in the repo.
